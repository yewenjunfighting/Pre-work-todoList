# Pre-work-todoList
## 实现的功能
> 1. 增加一个待办事项
> 2. 删除一个待办事项
> 3. 标记一个待办事项为已完成
> 4. 编辑一个待办事项的具体内容
> 5. 待办事项可以设置优先级
> 6. 待办事项可以设置expire date


## run
 ### rearEnd
 > 1.    cd rearEnd
 > 2.    python manage.py runserver 0.0.0.0:8000
 ### client
 > cd lient
 > 1.    npm install
 > 2.    npm start
